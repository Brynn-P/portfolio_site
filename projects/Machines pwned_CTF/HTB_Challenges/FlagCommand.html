Flag Command

This was a fun and quick web challenge

Please note that the flag is not visible here, learning is not done through copy and paste ;-)

Flag Command is a Hack The Box challenge classified as very easy

This was a fun quick web challenge that helps you remember some of the fundamentals.

For this challenge the IP address 154.57.164.61:31919 was used.

To navigate to this in the browser, simply use http:// with the IP address in your preferred browser.

When you navigate to the page, you are met with an interactive game.

The game is fun in its own way, and of course I had to give it a run but that waited until after I found the flag.

Now the first thing that caught my eye was that there is an API involved here. So I decided to have a look at what was coming through there.

When looking at the response section in Burp Suite, towards the end of the response we find an interesting secret option.

“Blip-blop, in a pickle with a hiccup! Shmiggity-shmack”

This must be how I get it done. We have found a leaked secret command!

In my excitement to get the command In, I clean forgot to start the game to be able to run the secret command.

Once we actually got the game going we managed to run the secret command, and what do you know....

Out falls a flag HTB{D********_*****_***_****__****_****_**_***_****k**}

This challenge was quite simple but truly a lot of fun to do and worked perfectly as a little warm up exercise.

Now that the challenge itself is done, its time to give the game a little bit of a try.

Alas I did not manage to beat the game, I was chased out of the forest by some angry bees

While I did not win the game, I did get the flag so I still walk away happy

Happy Hacking :-)
